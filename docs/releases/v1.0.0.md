# MetaFloat v1.0.0

MetaFloat 的第一个正式开源版本。

## 主要功能

- 连接现有 mihomo 控制器，并通过悬浮窗显示实时与累计流量。
- 优化主页连接状态和悬浮窗断线恢复。
- 支持打开 Zashboard，并提供多种下载源。
- 支持悬浮窗旋转边界修正、主题切换、加密 secret 存储和 Android 15 specialUse 前台服务。

## 安装提示

此前的测试包使用 Debug 密钥签名，无法直接覆盖安装。请先卸载测试版，再安装 `MetaFloat-v1.0.0.apk`。从本版本开始，后续正式版将沿用同一发布密钥。

请同时下载 `.sha256` 文件校验 APK 完整性。
